### What is Wedges?

Wedges is an open-source React UI library that combines the Lemon Squeezy's Wedges <a href="https://lemonsqueezy.com/wedges/" target="_blank">design system</a> with <a href="https://www.radix-ui.com/primitives" target="_blank" rel="noopener noreferrer">Radix UI primitives</a> and <a href="https://tailwindcss.com/" target="_blank" rel="noopener noreferrer">Tailwind CSS</a>. It's designed to simplify the process of building beautiful, functional and accessible UIs.

With Wedges, you get a range of pre-built, customizable components that are easy to use and integrate into your React projects. It also comes with full TypeScript support, ensuring type safety and developer-friendly experience.
