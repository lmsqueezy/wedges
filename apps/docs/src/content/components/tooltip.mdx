### Usage

```tsx
import { Tooltip } from "@lmsqueezy/wedges";
```

```tsx showLineNumbers
<Tooltip content="Tooltip Content" />
```

### API Reference

Extends the <a href="https://www.radix-ui.com/primitives/docs/components/tooltip#content" rel="nofollow noreferrer" target="_blank">Radix Tooltip Content</a> primitive and includes all of its props.
