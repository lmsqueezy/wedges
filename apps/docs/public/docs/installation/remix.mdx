### Setting up Remix

To set up a new Remix project, run the following command in your terminal:

```bash
npx create-remix@latest my-remix-app
cd my-remix-app
```

More information can be found in the <a href="https://remix.run/docs/en/main/start/quickstart" target="_blank" rel="nofollow noreferrer">official Remix Documentation</a>.

### Install Tailwind CSS

By default, Remix does not include Tailwind CSS, so we'll need to do that manually.

More information can be found in the <a href="https://tailwindcss.com/docs/guides/remix" target="_blank" rel='nofollow noreferrer'>official Tailwind CSS installation guide</a>.

Let's start by installing the Tailwind CSS:

### Configure Tailwind CSS

Update your `tailwind.config.ts` file to add the necessary configuration for Wedges:

```tsx {2,7,12-13} title="tailwind.config.js" showLineNumbers
import { wedgesTW } from "@lemonsqueezy/wedges";
import type { Config } from "tailwindcss";

export default {
  content: [
    "./app/**/*.{js,jsx,ts,tsx}",
    "node_modules/@lemonsqueezy/wedges/dist/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  darkMode: "class",
  plugins: [wedgesTW()],
} satisfies Config;
```

### Add the Directives

Create a `./app/tailwind.css` file and add the `@tailwind` directives for each of Tailwindâ€™s layers.

```css title="app/tailwind.css" showLineNumbers
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### Import the CSS File

Import the newly-created `tailwind.css` file in your `./app/root.jsx` file.

```tsx showLineNumbers title="root.tsx"
import stylesheet from "~/tailwind.css";

export const links: LinksFunction = () => [{ rel: "stylesheet", href: stylesheet }];
```
